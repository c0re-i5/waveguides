ℹ About

Paint different media on the canvas and watch waves interact with them in real time.

The simulation uses FDTD (Finite-Difference Time-Domain) — the same method from the waveguides repo scripts.

One equation governs everything:
∂²u/∂t² = c(x,y)² · ∇²u

The only difference between sound, light, and water waves is what c(x,y) means.

Controls:

🎨 Paint medium · 🧱 Draw walls · 〰️ Place oscillator · 💥 Drop pulse · ⌫ Erase

Space play/pause · R reset waves · C clear to preset · 1-4 select medium · Right-click erases
